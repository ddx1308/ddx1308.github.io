第六步：部署到Github上

1、登录github；

2、new repository


3、在_config.yml进行配置

4、 生成SSH添加到GitHub

打开你的git bash中，

成功生成了.ssh的文件夹。在你的电脑C:\Users\用户.ssh中找到id_rsa.pub这个文件。

然后在GitHub的settings中，找到SSH and GPG keys的设置选项，点击New SSH key，title随便起一个标题名称；

把你的id_rsa.pub用记事本的方式打开，里面的信息复制到Key里面去。

5、新建博客

6、把hexo部署到GitHub上

先安装hexo-deployer-git,就是部署的命令,这样你才能用命令部署到GitHub

然后

其中 hexo c 清除了你之前生成的东西，也就是hexo clean的缩写;
hexo g 生成静态文章，也就是hexo generate的缩写;
hexo d 部署，也就是hexo deploy的缩写。

部署成功就可以在http://你的仓库.github.io 这个网站看到你的博客了！！
